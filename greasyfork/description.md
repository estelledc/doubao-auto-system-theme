# 豆包网页版自动深色模式 / Doubao Web Auto Dark Mode

让豆包网页版聊天页跟随系统深浅色，并修复深色下侧栏历史、回答正文、代码块、输入框和弹窗的可读性。

## 为什么做

豆包网页版目前没有稳定可用的完整深色模式。晚上用的时候，常见问题是局部白底、黑字暗底、代码块看不清、左侧历史记录对比度太低。

这个脚本不是简单反色，而是：

- 先同步豆包页面的原生主题属性。
- 再针对聊天页里容易漏掉的区域做局部修补。
- 尽量保留图片、头像、品牌色和原有控件层级。

## 支持范围

- 只匹配：`https://www.doubao.com/chat/*`
- 推荐环境：Chrome + Tampermonkey
- 主题来源：系统深浅色设置
- 不提供手动强制开关

## 已重点适配

- 左侧历史记录和搜索框
- 聊天回答正文 Markdown
- 代码块、代码块顶栏、复制/展开按钮和语法高亮
- 底部输入框
- 下拉菜单、弹窗、toast、浮层

## 隐私说明

脚本不上传聊天内容，不请求外部接口，不做统计追踪。它只在浏览器本地修改豆包聊天页的主题属性和样式。

## 反馈问题

如果遇到白块、黑字、看不清或豆包改版后失效，请到 GitHub issue 反馈，并附上：

- 浏览器版本
- Tampermonkey 版本
- 系统主题
- 截图
- 控制台运行 `__doubaoThemeAudit()` 的输出

请先打码头像、姓名、企业信息和聊天隐私内容。
